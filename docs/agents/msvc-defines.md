# Compiler Defines (MSVC)

- `NOMINMAX` - Disables Windows min/max macros
- `WIN32_LEAN_AND_MEAN` - Minimal Windows headers
- `_CRT_SECURE_NO_WARNINGS` - Suppress CRT warnings
- `/MP` - Multi-processor build
- `/Z7` - Debug info (Debug config)
