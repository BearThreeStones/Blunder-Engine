# Reflection kernel is the first reflection milestone

The first implementation slice is the **Reflection kernel**: ClassDB; narrow Clang export for a small set of pilot types; PtrCall plus a small Variant path; API Blueprint; Object/`ObjectId` with Object property surface and projected Transform accessors; lazy Entity materialization for spatial writes; type-level lifecycle dispatch tables; and a C-ABI skeleton shaped for later script hosts. Explicitly out of scope for this milestone: shipping a .NET/nethost runtime, ALC hot reload, and a full ECS rewrite that replaces `SceneInstance`. Later phases consume the kernel (C# host, then hot reload, then broader ECS migration) without changing the ClassDB/C-ABI contract. Rejected alternative: bundling C# hot reload and full ECS replacement into the first reflection change.
